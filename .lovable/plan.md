

## 把品牌手册整合到官网的方案

将这份品牌手册的内容**分层级、分场景**地融入官网，让普通顾客看到温暖的品牌故事，让招商/加盟方能快速看到核心数据和合作信息。**不堆砌文字**，只挑选最有传播力的内容做视觉化呈现。

### 二、新增模块设计

**1. 战绩数据条 `AchievementsSection`**
单行横向 4-5 个大数字卡片，深色背景突出气势：
- 大众点评南京西路 / 静安区 **#1**
- 全网曝光 **300万+**
- 累计客流 **10万+ 人次**
- 月均收藏打卡 **1,000+**

**2. 双基因模式 `DualDNASection`**
左右对比布局："标准化 Book Off 模式" vs "街边设计师店"，中间汇聚成 BOOMER OFF logo，下方一句话点题。

**3. 空间体验装置 `ExperienceSection`**
2×2 网格，4 张大图 + 说明：
- 大通 Diatone 唱片机（听觉）
- 巨型 Gameboy（互动）
- 佐藤象店头（视觉地标）
- DIY 冰箱贴体验（参与）

**4. 翻筐乐 `FlipperFunSection`**
独立深色高亮区块，突出 **"6.9 元起"** 大字 + 寻宝筐图片，副标题"100+ 平价木筐 · 平均停留 45-90 分钟"。

**5. 全年龄段客群 `AudienceSection`**
5 个圆形人像/图标横排：儿童 · 青少年 · 都市白领 · 中年 · 老年，每个配一句典型场景。

**6. 媒体口碑 `PressSection`**
- SmartShanghai 报道引言（大字英文 + 中译）
- 下方 6 个高频关键词标签云（稀缺性 / 沉浸式 / 治愈 / 性价比 / 出片 / 有品位）

**7. 品牌矩阵 `BrandMatrixSection`**
7 张卡片横向滚动，每张标注：店型名 / 面积 / 一句话定位
（Vintage / Home / Hobby / Collection / Digital / NB / Plus）

**8. 招商合作 `PartnershipSection`**
- 标题"携手 BOOMER OFF"
- 标准店合作条件表（80-100㎡ / B1 或 1F / 销售扣点）
- 联系人潘瞻远 · 电话 · 加盟体验中心地址
- 主按钮"联系招商"

### 四、后台管理（Admin）扩展

为新增的可变内容模块加管理面板，避免硬编码：

| 新增 Manager | 管理字段 |
|---|---|
| **AchievementsManager** | 数据标签 / 数值 / 排序 / 是否显示 |
| **ExperienceManager** | 装置标题 / 描述 / 图片 / 排序 |
| **AudienceManager** | 客群名称 / 偏好品类 / 典型场景 / 图标 |
| **PressManager** | 媒体名 / 引言 / 译文 / 链接 |
| **BrandMatrixManager** | 店型名 / 面积 / 描述 / 排序 / 是否显示 |
| **PartnershipManager** | 合作条件项 / 联系人信息（单条记录） |

新增对应的 6 张 Supabase 表，全部带 RLS（公开可读 active 内容、管理员可改）。

---

### 五、技术细节

- 新模块全部用 Framer Motion 滚动淡入，保持站点风格
- 配色沿用品牌红 `#E30613` + 米白 + 深色对比块
- 字体沿用上次定下的系统字体栈（PingFang SC / Fredoka）
- 数据条的数字用 `framer-motion` 数字滚动动效
- 品牌矩阵卡片在移动端横向滚动（沿用 Gallery 的 snap 模式）
- 所有新增图片资源：先用占位渐变 + 图标，后续您可在后台上传真实图

---

### 六、不会做的事

- 不做 BOVAS 区块链溯源页面（手册标注"开发中"）
- 不放手机号到公开 Footer，敏感联系方式仅在「招商合作」区块中显示
- 不把整篇手册原文搬到网页（信息密度太高，会破坏品牌质感）

---

**预计交付**：约 8 个新组件 + 6 张新数据表 + 6 个后台管理面板 + 现有 5 个组件微调。所有内容均可在后台调整，您拿到新的店铺照片或更新数据后无需改代码。
